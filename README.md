# Algorithm-Journey

A collection of algorithm implementations and competitive programming solutions developed in C++.

## Project Overview
This repository serves as a personal archive for practicing C++ algorithms. It includes solutions to various problems, fundamental data structures, and algorithmic templates.

## Directory Structure
- **Source Files**: The root directory contains .cpp files for various algorithm exercises such as sorting, vector, and string operations.
- **.gitignore**: This file is configured to exclude binary executable files (*.exe) to maintain a clean repository.
